# About Me

Olá! I'm Forest, a programmer and artist coming from Brazil. I create all types of things, from pixel drawings, to applications, to compilers.

Outside of programming and art, I like a good game of chess, board games in general, videogames and fiction books.

## History

### Games

At 9, I started to wonder how games were made. I looked it up on YouTube, and found a Java tutorial.

#### Java

The tutorial was about how to make a tiny game: squares go through your screen in a straight line, and you click to destroy them before they go off-screen. I enjoyed the experience, and looked up how to make larger games.

_Now, in Java, I make libraries, as I started making apps in Rust and TypeScript, and games in GDScript._

#### C#

My search for how to make bigger games led me to discovering Unity, my first game engine. I made [little](https://github.com/shiftfox/lumina) [games](<(https://github.com/shiftfox/twenty-rockets)>) _(the latter was even before I learned Git, but I created a repo for it recently)_ much more easily.

_Now I don't really use C# that much, but I plan to use it again for larger Godot games or use Unity again. This is where I started making pixel art._

#### GDScript

[Certain things happened](https://unity.com/blog/unity-is-canceling-the-runtime-fee), and I switched from Unity to Godot. I learned GDSCript, and [made](https://github.com/shiftfox/scaleshot) [a lot of](https://github.com/shiftfox/turbulence) [little](https://github.com/shiftfox/box-of-death) [games](https://github.com/shiftfox/feathered-frenzy) for game jams, again.

_Now, GDScript is my primary language for games._

### Websites

At some point in this timeline, I wondered how _websites_ were made. I learned HTML, CSS and JavaScript, and made a website for my dad's scout group, without any frameworks.

Afterwards, I learned about the existence of Next.js, and later, Astro (as well as other _metaframeworks_ like SvelteKit and SolidStart). I rewrote the scout group website in Astro, had a much better experience and started making more personal projects (such as this website, also made in Astro!)

_Now, TypeScript is my most used language, as it's what you use in browsers and it's convenient having the same language in both the back and frontend._

### Low-level development

I learned about the Rust and Go programming languages, and started learning them. The projects I decided to use to learn these languages are not ready yet, but I will publish them as soon as possible!
